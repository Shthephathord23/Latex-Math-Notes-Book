This is a structured mathematical document project written in LaTeX. The documentation is split into the following files:

2. **[Document Architecture](document-architecture.instructions.md)** - File organization and structure patterns

## 🚀 Quick Start

1. **Build the document**: Use VS Code task "Build LaTeX Document" or run `pdflatex -interaction=nonstopmode main.tex`
2. **Add new content**: Follow the file organization patterns in [Document Architecture](document-architecture.instructions.md)
3. **Use custom notation**: Refer to [Notation System](notation-system.instructions.md) for available commands
4. **Follow mathematical style**: Use theorem environments from [Theorem Environments](theorem-environments.instructions.md)

## 📝 Key Principles

- **Modular architecture**: Each mathematical topic is organized in separate files and directories
- **Custom notation**: Use project-specific commands instead of raw LaTeX for consistency
- **Rigorous approach**: Follow established proof structure with detailed justifications
- **Standardized formatting**: Use custom theorem environments and notation commands

For detailed information on any aspect of the project, consult the relevant documentation file above.

Always maintain mathematical rigor and follow the established proof style with detailed justifications for each step.
